# AI-ML-Bootcamp-Sam

## 📌 Overview

This repository contains my learning materials, assignments, and practical implementations completed during the **AI & ML Bootcamp**. The primary focus of this repository is to understand the fundamentals of **Web Scraping** using Python and apply them through hands-on projects.

---

## 📂 Repository Contents

* **task-web-scrap.ipynb** – Jupyter Notebook containing the complete web scraping workflow.
* **webscrap-P1.pdf** – Project documentation/report.
* **README.md** – Project overview and usage instructions.

---

## 🎯 Objectives

* Learn the fundamentals of Web Scraping.
* Understand HTML page structure.
* Extract useful information from websites.
* Clean and organize scraped data.
* Practice data collection using Python.

---

## 🛠️ Technologies Used

* Python 3
* Jupyter Notebook
* BeautifulSoup (bs4)
* Requests
* Pandas

---

## 📚 Topics Covered

* Sending HTTP requests
* Parsing HTML documents
* Finding HTML elements
* Extracting text, images, and links
* Handling missing values
* Creating structured datasets
* Exporting data to CSV

---

## 🚀 How to Run

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/AI-ml-bootcamp-sam.git
   ```

2. Install the required libraries:

   ```bash
   pip install requests beautifulsoup4 pandas
   ```

3. Open the notebook:

   ```bash
   jupyter notebook
   ```

4. Run the cells in `task-web-scrap.ipynb`.
